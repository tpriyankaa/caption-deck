# 🗂️ Caption Deck

> *Pick a vibe, pull a card, post it.*

A fun social media caption generator built with pure HTML, CSS & JavaScript — no frameworks, no installs, just one file!

---

## ✨ Features

| Feature | Details |
|---|---|
| 🎴 Categories | General, Travel, Food, Selfie, Motivational, Funny, Pets, Fitness |
| 🎲 Random Picker | Never repeats the same caption twice |
| 📋 Copy Button | Copies caption + hashtags instantly |
| 💫 Animation | Smooth flip card effect |
| 📱 Responsive | Works on mobile & desktop |

---

## 🚀 How to Use

```bash
1. Open caption.html in any browser
2. Pick a category chip
3. Click "Pull a card"
4. Hit "Copy" and paste anywhere!
```
